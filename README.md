# Excel Sales Dashboard Project 📊

## Overview
This project contains a January month expenses dashboard created in Microsoft Excel. It includes data analysis and visualization of daily expenses.

## Features
- Expense tracking by category
- Payment mode analysis (UPI, Cash, Card)
- Monthly expense summary
- Clean and structured dashboard design

## Tools Used
- Microsoft Excel
- Data Analysis
- Data Visualization
